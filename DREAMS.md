# Memory Review Log

This file records memory consolidation passes: what was reviewed, what should be promoted, what should be archived, and what needs human confirmation.

Inspired by OpenClaw's "dreaming" concept, but intentionally manual at first. Later, a heartbeat job can append review candidates here before durable changes are promoted to `MEMORY.md`.

## Review Rules

- Promote only stable, repeated, or high-consequence facts to `MEMORY.md`.
- Keep raw observations in daily notes until they prove durable.
- Mark contradictions explicitly instead of silently overwriting memory.
- Ask for confirmation before storing sensitive, ambiguous, or potentially invasive inferences.
