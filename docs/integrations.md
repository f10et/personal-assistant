# Integrations

This file tracks intended integrations and constraints. Nothing here is implemented yet.

## Planned First

- Proton Calendar: reminders, planning, recurring commitments, and schedule review.
- Proton Mail: inbox triage, follow-up extraction, finance/admin document detection.
- Obsidian: notes import/export, cleanup, and knowledge base synchronization.

## Work Context

- Linear and Google Calendar matter for work, but the first project focus is private-life assistance.

## Future Automation

- Heartbeat job for daily planning and memory review.
- Weekly review job for open loops, finance admin, and relationship follow-ups.
- Document organizer for finance/tax/admin files.
- Subscription and bill tracker.

## Privacy Direction

Use the option that makes life materially better while keeping memory visible and reviewable. Prefer local-first storage for personal facts unless a hosted component clearly unlocks useful automation.
