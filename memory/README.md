# Memory Workspace

This directory holds working memory that is more detailed or less durable than `MEMORY.md`.

## Directories

- `daily/` - daily notes and short-term context.
- `inbox/` - raw captures waiting for review.
- `areas/` - maintained pages for important life areas.
- `reviews/` - weekly and monthly review outputs.

Promote only durable, high-signal items to `../MEMORY.md`.
