# Relationships

## Principle

Wife happy is a first-class planning concern.

## Assistant Role

- Track relationship follow-ups and commitments when the user asks.
- Help make time and attention visible in planning.
- Avoid creepy or over-inferential memory. Store relationship context only when it is useful and appropriate.

## Open Questions

- Important recurring dates.
- Preferred style for reminders.
- Which social follow-ups should be tracked.
